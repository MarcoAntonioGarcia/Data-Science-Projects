## 🇬🇧 English

### 📖 Overview
This repository is engineered to cover the **entire spectrum of Data Science**, from fundamental data analysis to advanced Artificial Intelligence solutions. The primary objective is to consolidate projects sourced from personal initiatives, online platforms (Udemy, Coursera), forums, and academic environments into a unified, professional-grade portfolio. 

> ⚠️ **Note**: This repository is an **ongoing project**. Modules and topics are continuously developed and refactored to conform to industry standards.

### 📂 Repository Structure & Roadmap

1. **[Data Collection & Cleaning](./2.-Data%20Cleaning%20&%20Preprocessing)**
   - Web scraping, APIs, database querying, data wrangling, and pre-processing.
2. **Data Analysis**
   - Exploratory Data Analysis (EDA), descriptive/inferential statistics, hypothesis testing.
3. **Data Visualization**
   - Data storytelling, dashboards using Power BI, Tableau, Matplotlib, Seaborn, Plotly.
4. **Databases (SQL & NoSQL)**
   - Relational (MySQL, PostgreSQL, Oracle) and Non-Relational (MongoDB, Firebase) database management.
5. **[Machine Learning](./6.-Machine%20Learning)**
   - Supervised and unsupervised learning, clustering, regression, classification algorithms.
6. **Deep Learning**
   - Neural networks architecture, CNNs, RNNs, transfer learning, frameworks like TensorFlow and PyTorch.
7. **Big Data & Cloud Pipelines**
   - Distributed computing with Apache Spark, Hadoop, and cloud data platforms (AWS, GCP, Azure).
8. **Automation & MLOps Workflow**
   - Workflow orchestration (Airflow, KNIME) and Python automation scripting.

## 🇪🇸 Español

### 📖 Descripción General
Este repositorio está diseñado para abarcar **todo el espectro de la Ciencia de Datos**, desde el análisis de datos fundamental hasta soluciones avanzadas de Inteligencia Artificial. El objetivo principal es consolidar proyectos provenientes de iniciativas personales, plataformas en línea (Udemy, Coursera), foros y entornos académicos en un portafolio unificado y de nivel profesional.

> ⚠️ **Nota**: Este repositorio es un **proyecto en desarrollo**. Los módulos y temas se desarrollan y refactorizan continuamente para cumplir con los estándares de la industria.

### 📂 Estructura del Repositorio y Hoja de Ruta

1. **[Recolección y Limpieza de Datos](./2.-Data%20Cleaning%20&%20Preprocessing)**
   - Web scraping, APIs, consultas a bases de datos, procesamiento y limpieza de datos estructurados/no estructurados.
2. **Análisis de Datos**
   - Análisis exploratorio de datos (EDA), estadística descriptiva/inferencial, pruebas de hipótesis.
3. **Visualización de Datos**
   - Narrativa de datos (storytelling), tableros interactivos usando Power BI, Tableau, Matplotlib, Seaborn, Plotly.
4. **Bases de Datos (SQL y NoSQL)**
   - Gestión de bases de datos relacionales (MySQL, PostgreSQL, Oracle) y no relacionales (MongoDB, Firebase).
5. **[Machine Learning](./6.-Machine%20Learning)**
   - Modelos de aprendizaje supervisado y no supervisado, algoritmos de clustering, regresión y clasificación.
6. **Deep Learning**
   - Arquitectura de redes neuronales, CNNs, RNNs, transfer learning, frameworks como TensorFlow y PyTorch.
7. **Big Data y Pipelines en la Nube**
   - Computación distribuida con Apache Spark, Hadoop y plataformas de análisis en la nube (AWS, GCP, Azure).
8. **Automatización y Flujos de Trabajo MLOps**
   - Orquestación de procesos (Airflow, KNIME) y scripting de automatización en Python.
